# Security Policy

## Supported Versions

The most recent published `4.x` release on Clojars receives security fixes. Both the `wire` and `wire-core` artifacts ship from the same `VERSION`, so a fix lands in both at once.

| Version | Supported          |
| ------- | ------------------ |
| 4.x     | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Please report security issues privately via [GitHub's "Report a vulnerability"](https://github.com/cleancoders/c3kit-wire/security/advisories/new) workflow, or by emailing the maintainer at gina@cleancoders.com. Do not open a public issue for security-sensitive reports.

We aim to acknowledge reports within 5 business days.
